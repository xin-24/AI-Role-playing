# 架构图说明

如果您无法查看README.md和架构设计文档中的Mermaid图表，可以参考以下文本描述：

## 整体架构图说明

```
用户浏览器 → 前端应用(React + Vite, 端口:3000/3001) → 后端服务(Spring Boot, 端口:8082)
                                                      ↘ 数据库
                                                      ↘ 七牛云服务 → ASR语音识别
                                                                   → TTS文本转语音
                                                                   → OSS对象存储
```

## 前端架构图说明

```
App组件 → 角色列表组件
       → 对话组件 → 语音输入模块
                 → 语音输出模块
                 → 消息显示模块
       → 角色管理组件
```

## 后端架构图说明

```
Spring Boot应用 → Controller层 → CharacterController
                              → ChatMessageController
                              → AsrController
                              → TtsController
               → Service层 → CharacterService
                         → ChatMessageService
                         → QiniuAsrService
                         → QiniuTtsService
               → Repository层 → CharacterRepository
                            → ChatMessageRepository
               → Config配置层 → Qiniu配置类
```

## 如何正确查看Mermaid图表

1. **使用VS Code**:
   - 安装"Mermaid Preview Enhanced"插件
   - 打开.md文件，右键选择"Mermaid: Preview Current Diagram"

2. **使用Typora**:
   - 下载并安装Typora
   - 直接打开.md文件即可查看图表

3. **使用在线编辑器**:
   - 访问 https://mermaid.live/
   - 将Mermaid代码粘贴到编辑器中查看

4. **转换为图片**:
   - 使用mermaid CLI工具将代码转换为PNG/SVG图片
   - 安装: `npm install -g @mermaid-js/mermaid-cli`
   - 转换: `mmdc -i input.mmd -o output.png`
